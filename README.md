# isPalaindrom solution (checking the the middel of the string)


```bash
npm install @adnen/ispalaindrom
```

## Test

```bash
npm run test
```

## Usage

```typescript
// TYPESCRIPT FILE
import  isPalindrome  from "@adnen/ispalaindrom";

const str : string = "racecar";
const result = isPalindrome(str);


console.log(result);
//  true
```
